# TAP-IN Game-to-Stripe Mapping

## Design Philosophy
Each stripe introduces skills → Games reinforce those skills → XP gates require practice before progression

---

## WHITE BELT: Trust Foundations

### Stripe 1: Trust Foundations
**Skills:** Understanding trust types, psychological safety basics
**Games:**
- **Confession Poker** (intro round) - Share something small, build comfort
- **Daily Dilemma** (trust scenarios) - Quick trust decisions
**Tools:**
- Journal (first entry: trust reflection)
- Inner Game (self-awareness baseline)
**XP Required to Unlock S2:** 75 XP (stripe gives 50, need 25 from games)

### Stripe 2: Practicing Vulnerability  
**Skills:** Strategic sharing, vulnerability edges
**Games:**
- **Confession Poker** (full game) - Deeper vulnerability practice
- **Trust Tower** (Level 1 only) - Low-risk sharing
**Tools:**
- Journal (vulnerability log)
- Mood Tracker (emotional awareness)
**XP Required to Unlock S3:** 100 XP

### Stripe 3: Self-Awareness
**Skills:** Triggers, stress responses, blind spots
**Games:**
- **Reaction Trainer** - Recognize emotional triggers
- **Daily Dilemma** (self-awareness focus) - How do you react?
**Tools:**
- Inner Game (deep dive)
- Energy Audit
- Box Breathing (stress management)
**XP Required to Unlock S4:** 125 XP

### Stripe 4: Trust Mastery
**Skills:** Repair trust, create safe environments
**Games:**
- **SBIR Trainer** (intro - appreciation mode) - Positive feedback
- **Confession Poker** (mastery round) - Full vulnerability
**Tools:**
- Weekly Review
- Morning Routine
**XP Required for Shark Tank:** 150 XP

## XP ECONOMY

### Earning XP
- Complete stripe content: 50-75 XP
- Play game (first time): 15-40 XP
- Replay game (diminishing): 5-15 XP
- Daily streak bonus: 10 XP
- Weekly challenge: 25 XP
- Shark Tank pass: 100 XP

### Spending XP (Gates)
- Unlock next stripe: Varies (see above)
- Unlock premium games: 50-100 XP
- Unlock tools: 25-50 XP

### Belt Totals Required
- White Belt Complete: 500 XP minimum
- Blue Belt Complete: 1,000 XP minimum
- Purple Belt Complete: 1,750 XP minimum
- Brown Belt Complete: 2,750 XP minimum
- Black Belt Complete: 4,000 XP minimum
