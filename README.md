# Tap-In App

A comprehensive leadership development platform with gamification, assessments, and team collaboration features.

## 🚀 Quick Start

### Local Development
```bash
# Clone the repository
git clone https://github.com/koenigermarco-tapin/Tap-In-App.git
cd Tap-In-App

# Start local server
python3 -m http.server 8000

# Open in browser
open http://localhost:8000
```

### Deployment
See [NETLIFY-SETUP-GUIDE.md](./NETLIFY-SETUP-GUIDE.md) for Netlify deployment instructions.

## 🎯 Features

- ✅ **Assessments:** Belt assessment, Leadership DNA, Work-Life Balance
- ✅ **Gamification:** XP system, achievements, celebrations
- ✅ **Team Features:** Create teams, join teams, sync data
- ✅ **Bilingual:** English and German support
- ✅ **PWA:** Offline support, service worker
- ✅ **Dark Theme:** Modern dark UI
- ✅ **Responsive:** Mobile-friendly design
