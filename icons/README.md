# App Icons for PWA

## Status: PLACEHOLDERS CREATED

This directory needs actual PNG icons in the following sizes:
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192) - **REQUIRED for PWA**
- icon-384.png (384x384)
- icon-512.png (512x512) - **REQUIRED for PWA**

## Temporary Solution:

The PWA will work without icons, but will show browser default icon.

## To Add Real Icons:

1. Create a 512x512 PNG with Tap-In branding
2. Use an online tool like https://realfavicongenerator.net/
3. Upload your 512x512 image
4. Download all sizes
5. Replace files in this directory
